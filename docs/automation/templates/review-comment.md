# Review Comment Template

Used by reviewer agents (Claude, Codex) when posting cross-review output to a PR. Mirrors `pr-review-rubric.md` exactly. CodeRabbit posts in its own format and does not use this template.

```markdown
## Cross-Review: {Role Name}

Generated by {agent} reviewing `{branch}`. Output follows `docs/automation/pr-review-rubric.md`.

## Findings

- **[severity] path:line** — finding with concrete impact and suggested fix.
- **[severity] path:line** — …

(If no findings: write `No findings for this role.`)

## Human Gates

- {Gate name from `responsibilities.md` Escalation list, or `None`.}

## Checks

- {Check that passed, failed, or was not run, with command if relevant.}

## Residual Risk

- {Remaining uncertainty after review.}
- {What you considered but could not fully verify.}

## Verdict

{approve | request-changes | needs-discussion}

(Only the integrator role uses the verdict words `mergeable | blocked | needs human decision | needs more review` — those describe the synthesis decision, not a single role's judgement.)
```

## Severity values

- **`blocker`** — must fix before merge. Not a stylistic preference.
- **`major`** — likely bug, regression, security risk, or spec mismatch.
- **`minor`** — correctness or maintainability issue worth fixing.
- **`note`** — non-blocking observation. Use sparingly; otherwise reviews get noisy.

## Rules

- One review block per role. If you serve multiple roles, post multiple blocks.
- Reference file:line. "There's a problem in the auth code" is useless; "auth.ts:42 — capability TTL is read but not validated" is actionable.
- Lead with findings. Reviewers who lead with summaries waste integrator time.
- Be honest about checks you could not run. The integrator needs to know what was actually verified vs. what was inferred.
- For minor/note severity: only include if the cumulative noise serves the reviewer. Many minor notes hide majors.
- Never approve your own work. If you (the agent) authored the PR, decline this role and recommend a different agent or a human.
